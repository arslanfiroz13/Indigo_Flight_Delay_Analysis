# ✈️ Indigo Flight Delay Analysis Dashboard

## 📌 Project Overview
This project analyzes flight delay data using **MySQL, Python, and Streamlit**.  
It provides an interactive dashboard to explore delays, routes, and trends.

---

## 🛠️ Tech Stack
- Python 🐍
- MySQL 🗄️
- Streamlit 📊
- Pandas

---

## 📊 Features
- Total Flights KPI
- Average Arrival & Departure Delay
- Top Delayed Flights
- Route Analysis
- Airport Delay Analysis
- Monthly Trends
- Distance vs Delay

---

## ⚙️ How It Works

- Data is stored in MySQL database
- SQL queries are executed to fetch flight data
- Python (Pandas) processes the data
- Streamlit displays interactive charts and KPIs
